# Política de Privacidade

**Zion Recruit - Plataforma de Recrutamento Inteligente**

**Versão**: 2.0  
**Data de Vigência**: Janeiro 2025

---

## 1. Introdução

A **Zion Assessment Technologies** ("nós", "nosso" ou "Zion") respeita sua privacidade e está comprometida a proteger seus dados pessoais. Esta Política de Privacidade explica como coletamos, usamos, compartilhamos e protegemos suas informações quando você utiliza nossa plataforma de recrutamento **Zion Recruit**.

### 1.1 Quem Somos

A Zion Assessment Technologies é uma empresa brasileira de tecnologia especializada em soluções de recrutamento inteligente e avaliação de talentos.

### 1.2 Abrangência

Esta política se aplica a:

- **Usuários da Plataforma**: Recrutadores, gestores de RH, administradores
- **Candidatos**: Pessoas que se candidatam a vagas
- **Visitantes**: Pessoas que acessam o Job Board público

## 2. Dados que Coletamos

### 2.1 Dados Fornecidos por Você

#### Dados de Cadastro (Usuários)

| Dado | Obrigatório | Finalidade |
|------|-------------|------------|
| Nome completo | Sim | Identificação |
| E-mail | Sim | Comunicação e login |
| Telefone | Não | Contato |
| Cargo/Função | Não | Personalização |
| Departamento | Não | Personalização |
| Foto de perfil | Não | Identificação |

#### Dados de Candidatos

| Dado | Origem | Finalidade |
|------|--------|------------|
| Nome completo | Candidato/Recrutador | Identificação |
| E-mail | Candidato/Recrutador | Comunicação |
| Telefone | Candidato/Recrutador | Contato |
| Currículo (arquivo) | Candidato/Recrutador | Avaliação |
| LinkedIn | Candidato/Recrutador | Verificação |
| GitHub | Candidato/Recrutador | Avaliação técnica |
| Portfolio | Candidato/Recrutador | Avaliação |
| Resultado DISC | Plataforma | Análise comportamental |
| Mensagens | Plataforma | Histórico de comunicação |

### 2.2 Dados Coletados Automaticamente

| Dado | Método | Finalidade |
|------|--------|------------|
| Endereço IP | Automático | Segurança |
| User Agent | Automático | Compatibilidade |
| Cookies | Automático | Funcionamento |
| Logs de acesso | Automático | Auditoria |
| Métricas de uso | Automático | Melhoria |

### 2.3 Dados de Pagamento

Processados exclusivamente pelo **Stripe** (certificado PCI-DSS):

- Número do cartão (não armazenamos)
- Data de validade (não armazenamos)
- CVV (não armazenamos)
- Dados de faturamento

---

## 3. Como Usamos Seus Dados

### 3.1 Finalidades Principais

| Finalidade | Base Legal | Dados |
|------------|------------|-------|
| Execução do contrato | Contrato | Todos os dados de cadastro |
| Processo seletivo | Contrato/Consentimento | Dados de candidatos |
| Comunicação | Contrato | E-mail, telefone |
| Segurança | Interesse legítimo | IP, logs, user agent |
| Melhoria do serviço | Interesse legítimo | Métricas anonimizadas |
| Marketing | Consentimento | E-mail |

### 3.2 Uso de Inteligência Artificial

Utilizamos IA para:

- **Parsing de currículos**: Extração automática de informações
- **Match scoring**: Cálculo de compatibilidade candidato-vaga
- **Análise DISC**: Interpretação de perfil comportamental
- **Geração de relatórios**: Síntese de informações

**Compromisso**: Dados processados por IA são tratados com privacidade, sem compartilhamento indevido com terceiros.

### 3.3 O que NÃO fazemos

- Vendemos dados pessoais
- Compartilhamos dados sem necessidade
- Utilizamos dados para finalidades não autorizadas
- Criamos perfis para fins discriminatórios

---

## 4. Compartilhamento de Dados

### 4.1 Com Quem Compartilhamos

| Destinatário | Finalidade | Dados |
|--------------|------------|-------|
| Empresas contratantes | Processo seletivo | Dados de candidatos |
| Stripe | Pagamentos | Dados de faturamento |
| OpenAI/Google | Processamento IA | Dados anonimizados/contextuais |
| Resend | Envio de e-mails | E-mail e conteúdo |
| Evolution API | WhatsApp | Mensagens |

### 4.2 Processadores de Dados

Todos os nossos processadores:

- Assinam contrato de processamento (DPA)
- Mantêm certificações de segurança
- Estão em conformidade com LGPD/GDPR

### 4.3 Situações Legais

Podemos compartilhar dados quando:

- Determinação judicial ou administrativa
- Proteção de direitos legais
- Prevenção de fraudes
- Emergências que ameacem a segurança

## 6. Cookies

| Tipo | Finalidade | Duração |
|------|------------|---------|
| **Essenciais** | Login, segurança | Sessão |

## 7. Segurança

### 7.1 Medidas Técnicas

| Medida | Descrição |
|--------|-----------|
| **Criptografia TLS** | Todas as comunicações |
| **Criptografia AES-256** | Dados sensíveis em repouso |
| **Hash bcrypt** | Senhas com salt |
| **WAF** | Firewall de aplicação |
| **Backups criptografados** | Recuperação de desastres |
| **Monitoramento 24/7** | Detecção de ameaças |

### 7.2 Sua Responsabilidade

- Mantenha suas credenciais seguras
- Não compartilhe sua conta
- Notifique-nos sobre atividades suspeitas
- Atualize seus dados regularmente

---

## 8. Retenção de Dados

### 8.1 Prazos

| Dado | Prazo | Justificativa |
|------|-------|---------------|
| Dados de conta | Duração da conta + 30 dias | Contrato |
| Candidatos | 24 meses após último contato | Interesse legítimo |
| Logs de auditoria | 5 anos | Obrigação legal |
| Dados financeiros | 5 anos | Obrigação fiscal |
| Backups | 90 dias | Recuperação |

### 8.2 Exclusão

Após os prazos, os dados são:

- Eliminados de forma segura, OU
- Anonimizados para fins estatísticos
