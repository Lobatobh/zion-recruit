## 1. Introdução

A **Zion Assessment Technologies** ("nós", "nosso" ou "Zion"), desenvolvedora da plataforma **Zion Recruit**, está comprometida com a proteção da privacidade e dos dados pessoais de todos os usuários da plataforma. Esta Política de Proteção de Dados descreve como coletamos, usamos, armazenamos e protegemos os dados pessoais em conformidade com a legislação aplicável.

### 1.1 Abrangência

Esta política aplica-se a:
- Todos os usuários da plataforma Zion Recruit
- Candidatos que se candidatam a vagas através da plataforma
- Empresas contratantes dos serviços
- Terceiros que interagem com nossos serviços

### 1.2 Legislação Aplicável

Estamos em conformidade com:
- **LGPD** - Lei Geral de Proteção de Dados (Lei nº 13.709/2018 - Brasil)
- **GDPR** - Regulamento Geral sobre a Proteção de Dados (UE 2016/679)
- **CCPA** - California Consumer Privacy Act (quando aplicável)
- **Marco Civil da Internet** (Lei nº 12.965/2014)

## 3. Dados Coletados

### 3.1 Dados de Usuários (Recrutadores/Administradores)

| Categoria | Dados | Finalidade | Base Legal |
|-----------|-------|------------|------------|
| **Identificação** | Nome, e-mail, telefone | Gestão de conta | Execução de contrato |
| **Acesso** | Senha (hash), 2FA | Segurança | Execução de contrato |
| **Profissionais** | Cargo, departamento | Personalização | Consentimento |
| **Técnicos** | IP, user agent, logs | Segurança | Interesse legítimo |

### 3.2 Dados de Candidatos

| Categoria | Dados | Finalidade | Base Legal |
|-----------|-------|------------|------------|
| **Identificação** | Nome, e-mail, telefone, foto | Processo seletivo | Execução de contrato |
| **Profissionais** | Currículo, LinkedIn, GitHub, portfolio | Avaliação | Consentimento |
| **Educacionais** | Formação, certificações | Avaliação | Consentimento |
| **Avaliação** | Resultados DISC, scores IA | Análise de perfil | Consentimento |
| **Comunicação** | Mensagens, e-mails trocados | Histórico | Execução de contrato |

### 3.3 Dados de Terceiros

| Categoria | Dados | Finalidade | Base Legal |
|-----------|-------|------------|------------|
| **Integrações** | Tokens de API, webhooks | Automação | Execução de contrato |
| **Pagamento** | Dados de faturamento (via Stripe) | Cobrança | Execução de contrato |

### 3.4 Dados Não Coletados

NÃO coletamos:
- Dados bancários completos (processados via Stripe)
- Número de CPF/RG (exceto quando necessário para contratação)
- Dados de saúde
- Dados de crianças (menores de 18 anos)
- Dados sensíveis sem consentimento explícito

## 4. Finalidades do Tratamento

### 4.1 Finalidades Principais

1. **Execução de Contrato**
   - Prestação dos serviços de recrutamento
   - Gestão de candidatos e vagas
   - Comunicação entre partes

2. **Consentimento do Titular**
   - Avaliações comportamentais (DISC)
   - Integração com redes sociais
   - Comunicações de marketing

3. **Interesse Legítimo**
   - Segurança da plataforma
   - Logs de auditoria
   - Melhoria de serviços
   - Prevenção de fraudes

4. **Obrigações Legais**
   - Resposta a ordens judiciais
   - Retenção de dados fiscais

### 4.2 Uso de IA

Os dados dos candidatos são processados por sistemas de Inteligência Artificial para:
- Parsing de currículos
- Cálculo de match score
- Análise de perfil DISC
- Geração de relatórios

**Compromisso**: Os algoritmos de IA são auditados regularmente para evitar discriminação e viés.

## 5. Compartilhamento de Dados

### 5.1 Compartilhamento Permitido

| Destinatário | Finalidade | Dados |
|--------------|------------|-------|
| **Empresa Contratante** | Processo seletivo | Dados de candidatos |
| **Stripe** | Processamento de pagamentos | Dados de cobrança |
| **OpenAI/Google** | Processamento de IA via API | Dados anonimizados |
| **Resend** | Envio de e-mails | E-mail e conteúdo |
| **Autoridades** | Obrigações legais | Conforme solicitado |

### 5.2 Compromissos com Terceiros

Todos os terceiros processadores de dados:
- Assinam DPA (Data Processing Agreement)
- Mantêm certificações de segurança (SOC 2, ISO 27001)
- Estão localizados em jurisdições com proteção adequada

### 5.3 Transferência Internacional

Quando necessário, a transferência internacional de dados é realizada:
- Com cláusulas contratuais padrão (SCC)
- Com garantias adequadas de proteção
- Em conformidade com decisões de adequação

## 6. Retenção de Dados

### 6.1 Prazos de Retenção

| Tipo de Dado | Prazo | Justificativa |
|--------------|-------|---------------|
| **Dados de conta** | Duração da conta + 30 dias | Execução de contrato |
| **Candidatos** | 24 meses após último contato | Interesse legítimo |
| **Logs de auditoria** | 5 anos | Obrigação legal |
| **Dados financeiros** | 5 anos | Obrigação fiscal |
| **Backups** | 90 dias | Recuperação de desastres |

### 6.2 Anonimização

Após os prazos de retenção, os dados são:
- Anonimizados para fins estatísticos, OU
- Eliminados de forma segura

### 6.3 Exclusão Antecipada

O titular pode solicitar:
- Exclusão de dados (quando não houver obrigação legal)
- Portabilidade de dados
- Anonimização

| Medida | Implementação |
|--------|---------------|

| **Rate Limiting** | Proteção contra abuso |
